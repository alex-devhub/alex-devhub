## Featured Work

### Patterna
Configurable AI knowledge assistant built with React, TypeScript, a server-side API route, and the Gemini API.

🔗 Repository: https://github.com/alex-devhub/patterna-ai-assistant

The project demonstrates:
- Secure API integration where the browser never receives the Gemini key
- A configurable knowledge base for FAQs, support content, onboarding notes, or portfolio data
- Structured AI responses with confidence, sources, and suggested follow-up questions
- Guardrails to avoid inventing unsupported information
